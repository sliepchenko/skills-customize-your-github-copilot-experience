# 📘 Assignment: Games in Python
## 🎯 Objective
Build a simple Hangman game in Python where players guess letters to reveal a hidden word, with a limited number of incorrect guesses allowed.
## 📝 Tasks
### 🛠️ Implement the Hangman Game
#### Description
Complete the provided starter code (`starter-code.py`) to create a fully functional Hangman game. The game should randomly select a word, allow players to guess letters, track correct and incorrect guesses, and end when the word is guessed or the maximum incorrect guesses are reached.
#### Requirements
Completed program should:
- Randomly select a word from the predefined list
- Display the current state of the word with underscores for unguessed letters
- Prompt the user for a single letter guess
- Update the display based on correct or incorrect guesses
- Track and display the number of incorrect guesses remaining
- End the game when the word is fully guessed or when maximum incorrect guesses (e.g., 6) are reached
- Display a win or lose message at the end
