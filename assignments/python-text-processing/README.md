# 📘 Assignment: Python Text Processing

## 🎯 Objective

Learn how to work with strings, file I/O, and text manipulation techniques in Python. By the end of this assignment, you'll be able to read files, process text data, and generate formatted output.

## 📝 Tasks

### 🛠️ Read and Display File Contents

#### Description
Create a Python script that reads a text file and displays its contents in a formatted way. This task introduces file I/O operations and basic string manipulation.

#### Requirements
Completed program should:

- Open and read a text file
- Parse the content line by line
- Display the content with line numbers
- Handle file not found errors gracefully

### 🛠️ Count Words and Characters

#### Description
Extend your script to analyze the text by counting total words, characters, and unique words. This task practices string methods and data structures.

#### Requirements
Completed program should:

- Count total characters (excluding whitespace)
- Count total words
- Count unique words (case-insensitive)
- Display a summary report

### 🛠️ Find and Replace Text

#### Description
Build a function that finds specific words in the file and allows replacing them with new text. This task covers string searching and replacement operations.

#### Requirements
Completed program should:

- Search for a specific word/phrase (case-insensitive)
- Replace all occurrences with a new word/phrase
- Save the modified text to a new file
- Display how many replacements were made
